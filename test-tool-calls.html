Message Assistant avec Tool Calls

Voici un exemple de ce que devraient ressembler les tool calls :

search_web
{ "query": "latest AI developments", "max_results": 5 }
get_weather
{ "location": "Paris", "unit": "celsius" }

Structure des données attendue :

{
  "role": "assistant",
  "content": "Voici les informations demandées...",
  "tool_calls": [
    {
      "id": "call_123",
      "type": "function",
      "function": {
        "name": "search_web",
        "arguments": "{\"query\": \"latest AI developments\", \"max_results\": 5}"
      }
    }
  ],
  "tool_results": [
    {
      "tool_call_id": "call_123",
      "name": "search_web",
      "content": "Résultats de recherche...",
      "success": true
    }
  ]
}

Instructions de test :

  1. Ouvrir la console du navigateur (F12)
  2. Aller sur le chat
  3. Envoyer un message qui déclenche des tool calls
  4. Vérifier les logs dans la console
  5. Vérifier l'affichage des tool calls dans l'interface
